

 

 # 🔐 CipherOS

"Built for the ones who know."

 

🧠 Overview

CipherOS is a powerful, Ubuntu-based Linux distribution engineered for cybersecurity professionals, gamers, creators, and power users.

It combines performance, privacy, and full system control into a single, clean and highly customizable experience.

Whether you're pentesting networks, running games at peak performance, or building your next project — CipherOS adapts to you.

 

🚀 Key Features

🎮 Gaming Ready

Preconfigured Steam, Lutris, Heroic Launcher

Full Wine + Proton compatibility

Performance tools: GameMode, MangoHUD, Gamescope

Vulkan + DXVK support out of the box

Optimized with XanMod Kernel for low latency

 

🔐 Cybersecurity Suite

A complete toolkit available from day one:

Recon:  nmap ,  masscan ,  theHarvester ,  gobuster 

Exploitation:  metasploit ,  sqlmap ,  hydra 

Wireless:  aircrack-ng ,  kismet ,  wireshark 

Password auditing:  hashcat ,  john 

Web testing:  burpsuite ,  nikto ,  dirb 

Anonymity:  tor ,  onionshare ,  proxychains 

 

🎨 Creative Tools

3D & Design: Blender, FreeCAD

Video: Kdenlive, DaVinci Resolve (auto-install script)

Image Editing: GIMP, Krita, Inkscape

Audio: Ardour, Audacity

 

⚡ Performance Optimized

CipherOS is tuned for speed and efficiency:

 zram  for compressed memory

 ananicy-cpp  for process prioritization

 preload  for smarter app launching

 irqbalance ,  thermald 

Custom kernel &  sysctl  tweaks

 

🔒 Privacy First

Privacy is not optional — it's built-in:

Ubuntu telemetry removed

DNS over HTTPS (dnscrypt-proxy)

MAC address randomization

Hardened UFW firewall (deny incoming by default)

App isolation via Firejail + AppArmor
